ABOUT

> MISSION OBJECTIVE

Project Slushie Lab is a high-speed flavor distribution interface developed for the Hack Club Slushies grant track. Total elimination of flavor indecision through randomized slushie allocation. Every spin is final. Every flavor is fate.

> HARDWARE ENVIRONMENT

Instance hosted on Ubuntu 24.04 LTS (Noble Numbat). Backend architecture utilizes Python-based Flask for routing and template rendering. Frontend is pure Canvas API + vanilla JS — zero framework bloat, maximum velocity.

> CORE TECHNOLOGIES
Flask / Jinja2
Canvas API
Web Audio API
CSS Variables
LocalStorage
Vercel Edge
> FEATURES DEPLOYED

Quantum Mixer — blend any two flavors at the molecular level. Flavor Passport — every spin, logged and timestamped. Custom Lab — build your own wheel, save your own flavors. Dynamic theming — the entire UI reacts to your winning flavor's hex code, including the text selection color. Confetti engine, audio tick system, keyboard shortcuts, live spin streaks.
